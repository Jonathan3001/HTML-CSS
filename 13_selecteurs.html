Aide mémoire des sélecteurs CSS déja vus
  • * sélectionne TOUS les éléments de la page html
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les éléments ayant le nom de mon id "mon-id-unique"
  • div, article, aside sélectionne toute les balise "div", "article" ET "aside"
  • nav a sélectionnes tous les liens "a" inclus/imbriqués dans la balise nav
  • nav a: hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris
Propriétés CSS utilisées dans cette page
Décrivez leur usage et leurs valeurs possibles
  1. background: couleur de font de l'élément
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. height: hauteur de l'élément
  7. left: position à partir de la gauche de la fenêtre
  8. line-height: hauteur de l'interligne
  9. margin: marges extérieures
  10. padding: marges intérieures
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du texte
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour tous les navigateurs est supprimé)
  14. top: position à partir du haut de la fenêtre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menu)
  16. width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em,...)
